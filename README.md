# TsukiHub - Web Personal

Web personal con React para gestionar Juegos, Peliculas, Pokedex, Gacha, GameBoy y TTS.

## Descripcion

Aplicacion web completa que incluye:

- **Inicio**: Dashboard con estadisticas de usuarios y logros
- **Juegos**: Gestion de videojuegos
- **Pelis**: Gestion de peliculas y series
- **Pokedex**: Coleccion de Pokemons por usuario y region
- **Gacha**: Sistema de gacha con diferentes banners
- **GameBoy**: Tutorial sobre el uso de la GameBoy
- **TTS**: Tutorial y validador de texto para TTS de Twitch
